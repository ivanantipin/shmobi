install flutter sdk https://flutter.dev/docs/get-started/install/linux
install flutter plugin into idea
create in idea new module from existing - grevel folder
execute 'grevel/gen.sh' to generate protos
make sure you have android facet to test application on grevel module
download on of images of phones for android via tools->android->sdk manager
run android emulator via tools->android->avd manager
start main.dart which is in grevel/lib/
